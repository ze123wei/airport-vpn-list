# airport-vpn-list

机场 VPN 平台整合目录，整理公开可查的免费、试用、收费及客户端型服务资料。

> 本仓库用于信息整理，不构成推荐、性能承诺或购买建议。  
> 套餐、价格、活动、链接和服务状态可能变化，请以对应页面当前信息为准。

## 在线目录

[![进入完整平台目录](https://img.shields.io/badge/进入完整平台目录-2167B5?style=for-the-badge&logo=githubpages&logoColor=white)](https://你的用户名.github.io/airport-vpn-list/)

完整页面包含：

- 当前已整理的平台目录
- 平台分类与访问页面入口
- 套餐资料与公开说明
- 新手指南：机场、节点、订阅、协议、流量倍率等基础介绍
- 平台资料跳转与筛选功能

## 当前发布范围

目前优先展示已经完成链接和资料整理的平台；其余平台会在后续核对、补充资料后逐步加入。

## 使用说明

1. 先进入在线目录查看平台分类和资料状态。
2. 点击“访问页面”查看对应平台当前页面。
3. 点击“详细资料点链接”跳转到目录内的套餐资料。
4. 价格、套餐、活动和可购买状态可能变化，查看时请注意最后核验日期。

## 目录原则

- 仅记录公开可查的信息。
- 不以页面宣传、节点名称或套餐名称作为独立性能结论。
- 不收录账号、订阅配置、邀请码或其他敏感信息。
- 平台被收录不代表稳定、安全或适合所有人。
